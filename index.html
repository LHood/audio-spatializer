Audio Spatializer
What is This?

Enter YouTube Video URL

This application downloads the given YouTube video and runs an algorithm on it to gather information about the beat and structure. It then streams the audio to the application and uses the Resonance Audio API to move the audio source in 3D space around the listener, creating an immersive listening experience. This web application is best used with stereo headphones or earphones and works best on dubstep or songs with a strong distinguishable beat and bass line.
